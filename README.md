# Fitness CLI

## ⚙️ Funksjonaliteter

FitnessCLI er et terminal-basert program i C# som hjelper deg å oppnå dine fitnessmål. Et par av funksjonene som du kan gjøre med dette verktøyet:

- Få et sett av øvelser per muskelgruppe, og så kombinere det til én (trenings)økt. <br> 
Tenk f. eks. på å få generert en brystøkt, som også inkluderer sett som trener dine triceps og skuldre.

- Lage en profil, og regne ut ting som din BMI ved hjelp av den.
